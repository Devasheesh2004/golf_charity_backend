# ImpactGolf: Monthly Charity Subscription Platform

## Project Overview
ImpactGolf is a specialized SaaS platform designed to merge the competitive spirit of golf with philanthropic impact. It allows golfers to turn their performance into charitable contributions while participating in transparent, monthly prize draws.

---

## 🛠 Technical Stack
- **Frontend**: Next.js 14, Tailwind CSS, Lucide Icons, React Context API.
- **Backend**: Node.js, Express, TypeScript.
- **Database**: Supabase (PostgreSQL with Real-time synchronization).
- **Payments**: Stripe (Stripe Checkout & Webhooks for subscription lifecycle).
- **Emails**: SendGrid API (REST protocol for reliability in serverless/cloud environments).
- **Authentication**: JWT-based secure sessions with OTP (One-Time Password) verification.

---

## 🔄 User Workflow
### 1. Onboarding & Security
- Users sign up with verified credentials.
- Multi-Step Login: The system uses an **OTP-based Security Check** via SendGrid to ensure account integrity.

### 2. The Subscription Model
- Integration with Stripe allows users to choose between **Standard** and **Pro** tiers with monthly or annual billing.
- Subscriptions are handled via **Stripe Webhooks**, ensuring that access is granted only after a successful transaction.

### 3. Philanthropy & Impact Selection
- Every subscriber selects a **Charity Recipient** from a curated list.
- **Voluntary Increase**: Users have a dedicated control panel to increase their impact (from a 10% base up to 100%), directly amplifying the contribution generated by their subscription.

### 4. Competitive Engagement (Score Tracking)
- Users submit their official Stableford scores to the platform.
- The history tracks the latest 5 entries to maintain a fair performance record for the upcoming draw.

### 5. Monthly Prize Draws
- The platform hosts a **Centralized Draw System**.
- **Winning Mechanism**: Each month,winning numbers are generated. Users who match the most numbers win a share of the accumulated prize pool.
- **Rollover Logic**: If a jackpot tier is not matched, the funds automatically roll over to the next month to increase engagement.

---

## 👑 Admin Control Panel
The platform includes an advanced administrative interface for operational management:
- **User Management**: Monitor subscriber growth and account statuses.
- **Charity Management**: Update featured charities and track total distributions.
- **Draw Simulation**: Admins can run "Simulation Mode" to preview winner distribution and rollover amounts before officially publishing results.
- **Prize Verification**: Manual review of score screenshots/proof uploaded by winners to ensure transparency.

---

## 🛡 Security & Deployment
- **CORS Protection**: Access is restricted to trusted frontend domains.
- **Environment Isolation**: Production keys (Stripe, SendGrid, Supabase) are managed via Render's secure environment variable layer.
- **Webhook Integrity**: Stripe endpoints utilize cryptographic signature verification (whsec) to prevent spoofing.

---

## 🏅 Submission highlights
- **Responsive Ecosystem**: Fluid design optimized for mobile-first user interaction.
- **API Transparency**: High-traffic areas utilize lazy-loading and debounced hooks for smooth performance.
- **Reliability**: Migrated from standard SMTP to REST-based API dispatch to guarantee 99.9% email delivery in cloud environments.
